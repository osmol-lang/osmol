# Contributing to Osmol

Welcome — you found the house early.

- Questions and ideas: open an Issue.
- Language changes (grammar/semantics): open an RFC at osmol-lang/rfcs.
- Code changes: open a Pull Request; keep it small; the conformance
  examples (dinner.osmol, spam.osmol) must still behave identically.
- One law of the codebase: the word `send` may appear only in the
  tombstone machinery.
